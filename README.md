# Telegram бот студии звукозаписи (aiogram + SQLite)

Бот для записи в студию: календарь через inline-кнопки, слоты по 3 часа, оплата по реквизитам, подтверждение “✅ Я оплатил”, админ-панель, публикация расписания в канал и автонапоминания за 24 часа.

## Возможности

- Запись на ближайшую неделю (inline-календарь).
- 8 слотов по 3 часа (можно управлять слотами в админке).
- Один пользователь = одна активная запись.
- Обязательная проверка подписки на канал перед записью (`getChatMember`).
- Отмена записи пользователем (слот снова доступен).
- Админ-панель: рабочие дни, слоты, закрытие дня, отмена записей, просмотр расписания.
- Уведомление админу + публикация расписания в канал.
- Автонапоминание за 24 часа (APScheduler) + восстановление после перезапуска.
- Разделы “Прайсы” и “Оборудование и Фото Студии” (с фото из локальных файлов).

## Установка

```bash
cd C:\Users\zaggo\recording_studio_bot_v2
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Настройка

1) Скопируйте `.env.example` → `.env`
2) Заполните минимум:
- `BOT_TOKEN`
- `ADMIN_ID`
- `CHANNEL_ID`, `CHANNEL_LINK`
- `SCHEDULE_CHANNEL_ID` (можно тот же канал)
- `PAYMENT_DETAILS`

## Запуск

```bash
python bot.py
```

## Команды

- `/start` — главное меню
- `/admin` — админ-панель (только `ADMIN_ID`)
